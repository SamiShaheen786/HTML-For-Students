📘 Introduction to HTML

HTML (HyperText Markup Language) is the foundation of every webpage.

What HTML Does

  • Structures content — headings, paragraphs, lists
  • Embeds media — images, audio, video
  • Creates hyperlinks between pages
  • Builds forms for user input
HTML vs CSS vs JavaScript
HTML = Structure  |  CSS = Style  |  JS = Behaviour

Your First HTML Tag
