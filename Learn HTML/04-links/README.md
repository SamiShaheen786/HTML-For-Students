# 04 - Links (Anchor Tag)

## Files
| File | What it covers |
|------|---------------|
| `index.html` | External, internal, email, phone, anchor (jump) links |

## Syntax
```html
<a href="URL">Link Text</a>
```

## Key Attributes
- `href` — where the link goes
- `target="_blank"` — opens in new tab
- `rel="noopener noreferrer"` — security when using target="_blank"
