## What is Semantic HTML?
A semantic tag clearly describes its meaning to the browser AND the developer.

```html
<!-- Non-semantic (meaningless) -->
<div id="nav">...</div>
<div id="footer">...</div>

<!-- Semantic (meaningful) -->
<nav>...</nav>
<footer>...</footer>
```

## Why Semantic HTML Matters
1. Screen readers understand the page structure
2. Search engines rank semantic pages better (SEO)
3. Code is easier to read and maintain
