ARIA Roles and Attributes

ARIA = Accessible Rich Internet Applications. Use ARIA when HTML alone cannot describe the purpose of an element.

Rule: Always prefer correct semantic HTML over ARIA. Only use ARIA when no HTML tag exists for what you need.

aria-label

Use when a button/link has no visible text (like an icon button).

aria-hidden

⭐⭐⭐⭐ 4 out of 5 stars

Home | About | Contact

aria-expanded

Use on buttons that toggle content open/closed (menus, accordions).

role attribute

lang attribute (on html tag)

The lang="en" on the html tag tells screen readers what language to use. This page has lang="en" so screen readers use English pronunciation.

If part of your page is in another language, use lang on that specific element:

یہ اردو میں ہے۔ (This is in Urdu)

هذا باللغة العربية (This is in Arabic)
