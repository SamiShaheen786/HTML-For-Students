Alt Text for Images

The alt attribute describes an image for screen readers and also shows as text if the image fails to load.

Bad Alt Text Examples (do NOT do this)

alt="image" — completely meaningless


alt="img_001.jpg" — file name is not a description


alt="image of a person" — screen reader already says "image", so this is redundant

Good Alt Text Examples (do this)

alt="Snow-covered mountains at sunrise with orange sky" — describes what you see


alt="" (empty) — decorative images that add no information should have empty alt

Form Accessibility: Always Use label

Every input needs a label. Screen readers read the label aloud when user focuses the input.
