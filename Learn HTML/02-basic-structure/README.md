# 02 - Basic HTML Structure

Every HTML file must follow this exact skeleton.

## Files
| File | What it covers |
|------|---------------|
| `index.html` | The full skeleton — DOCTYPE, html, head, body |

## The Skeleton
```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <title>Page Title</title>
  </head>
  <body>
    <!-- visible content here -->
  </body>
</html>
```

## What Each Part Does
- `<!DOCTYPE html>` — tells browser this is HTML5
- `<html>` — root of the page
- `<head>` — metadata (not visible on page)
- `<body>` — everything the user sees
