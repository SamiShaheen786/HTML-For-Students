Meta and Head Tags

All the important meta information for this page is inside the <head> tag above. Right-click this page and click View Page Source to see all the meta tags in action.

What the head tag contains:

  • charset — character encoding (supports all languages)
  • viewport — makes page mobile-friendly
  • description — shown in Google search results
  • keywords — hints to search engines
  • author — who wrote the page
  • og tags — controls social media sharing preview
  • title — browser tab text and Google headline
  • link favicon — small icon in the browser tab
  • link stylesheet — connects your CSS file
  • script — connects your JavaScript file

Key Rule

Everything in <head> is for the browser and search engines. NOTHING in <head> is directly visible on the webpage itself (except the title which shows in the tab).
