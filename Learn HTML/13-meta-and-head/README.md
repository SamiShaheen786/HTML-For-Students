# 13 - Meta & Head Tags

## Files
| File | What it covers |
|------|---------------|
| `index.html` | charset, viewport, description, keywords, author, og tags, favicon, linking CSS & JS |

## Key Meta Tags
```html
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <meta name="description" content="Page description shown in Google search results" />
  <meta name="keywords" content="html, css, web, tutorial" />
  <meta name="author" content="Ali Hassan" />
  <title>My Page Title</title>
  <link rel="icon" href="favicon.ico" />
  <link rel="stylesheet" href="style.css" />
  <script src="app.js" defer></script>
</head>
```
