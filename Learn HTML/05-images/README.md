# 05 - Images

## Files
| File | What it covers |
|------|---------------|
| `index.html` | img tag, src, alt, width, height, figure, figcaption |

## Syntax
```html
<img src="path/to/image.jpg" alt="description" width="300" />
```

## Key Attributes
- `src` — path to image (local or URL)
- `alt` — description for screen readers (NEVER skip this)
- `width` / `height` — size in pixels
