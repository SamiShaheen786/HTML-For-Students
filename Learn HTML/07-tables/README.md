# 07 - Tables

## Files
| File | What it covers |
|------|---------------|
| `basic-table.html` | table, tr, th, td, thead, tbody, tfoot |
| `spanning.html` | colspan (merge columns), rowspan (merge rows) |

## Syntax
```html
<table>
  <thead>
    <tr> <th>Name</th> <th>Age</th> </tr>
  </thead>
  <tbody>
    <tr> <td>Ali</td>  <td>21</td>  </tr>
  </tbody>
</table>
```

## Key Tags
- `table` ‚Äî the whole table
- `tr` ‚Äî table row
- `th` ‚Äî table header cell (bold by default)
- `td` ‚Äî table data cell
- `thead` ‚Äî header section
- `tbody` ‚Äî body section
- `tfoot` ‚Äî footer section
