Ordered List (ol)

ol = ordered list. Items are numbered. ORDER matters.

  1. Wake up
  2. Brush teeth
  3. Have breakfast
  4. Start studying HTML

type = "1" (default - numbers)

  1. First
  2. Second
  3. Third

type = "A" (uppercase letters)

  1. Option A
  2. Option B
  3. Option C

type = "a" (lowercase letters)

  1. Sub item a
  2. Sub item b
  3. Sub item c

type = "I" (uppercase Roman numerals)

  1. Chapter One
  2. Chapter Two
  3. Chapter Three

type = "i" (lowercase Roman numerals)

  1. Step one
  2. Step two
  3. Step three

start = "5" (start from 5)

  1. Item Five
  2. Item Six
  3. Item Seven

reversed (countdown)

  1. Third place
  2. Second place
  3. First place
