# 06 - Lists

## Files
| File | What it covers |
|------|---------------|
| `unordered-list.html` | ul, li — bullet lists |
| `ordered-list.html` | ol, li — numbered lists, type attribute |
| `nested-list.html` | list inside a list |

## Syntax
```html
<!-- Unordered -->
<ul>
  <li>Item</li>
</ul>

<!-- Ordered -->
<ol>
  <li>Item</li>
</ol>
```
