# 10 - ID and Classes

## Files
| File | What it covers |
|------|---------------|
| `index.html` | class attribute, id attribute, multiple classes, difference between id and class |
| `style.css` | CSS to demonstrate how id and class selectors work |

## Key Difference
| class | id |
|---|---|
| Can be used on MANY elements | Must be UNIQUE on a page (only one element) |
| Selected in CSS with `.classname` | Selected in CSS with `#idname` |
| One element can have multiple classes | One element should have only one id |
| Use for styling groups | Use for unique elements or anchor links |

## Syntax
```html
<p class="text1">Paragraph with class</p>
<p id="main-title">Unique element with id</p>
<p class="text1 colored">Multiple classes on one element</p>
```
