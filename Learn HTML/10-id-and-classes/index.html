class attribute

Both paragraphs below share class="text1" so they get the same style (blue color).

Multiple Classes on One Element

You can put more than one class on an element. Separate with a space.

This paragraph has class="text2 colored-text" — it gets green color from text2 AND yellow background from colored-text.

This paragraph only has class="text2" (no yellow background).

id attribute

Each paragraph below has a UNIQUE id. Each gets its own individual style.

One Paragraph (id = one) — styled red in CSS

Two Paragraph (id = two) — styled purple in CSS

WRONG: Duplicate id (do not do this!)

Welcome (h1 with id="heading")

Welcome (h2 with same id="heading")

CORRECT: Use class for multiple elements

When you want the same style on multiple elements, use a class not an id.

Welcome (h3 with class="hi")

Welcome (h4 with class="hi")

Summary: class vs id

Use CLASS (.name) when: multiple elements share the same style.

Use ID (#name) when: it is one unique element, OR you need an anchor link target.
