Inline and Block Elements

Block Elements

Block elements always start on a NEW line and take the FULL width available.

I am a div (Block Element) - I take full width
I am another div - I start on a new line

I am a p tag (Block Element) - also starts on new line

I am an h3 (Block Element)

Other block elements: div, p, h1-h6, ul, ol, li, table, form, header, footer, section

Inline Elements

Inline elements stay on the SAME line. They only take as much width as their content.

I am span (Inline) I am also span (Inline) We all stay on the same line!

Google Link (Inline) GitHub Link (Inline)

Other inline elements: span, a, img, strong, em, b, i, u, mark, label, input

Practical Example

Hi I Am Back and today we are learning about inline and block elements in HTML.

This p is INSIDE a div. Both are block elements.

The div wraps them like a container.

This sentence has a red word in the middle using a span tag (inline).

Key Rule

You CAN put inline elements inside block elements. (OK)

You should NOT put block elements inside inline elements. (Wrong)

Correct: span inside p
