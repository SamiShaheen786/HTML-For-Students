# 09 - Inline & Block Elements

## Files
| File | What it covers |
|------|---------------|
| `index.html` | block vs inline elements with CSS to show the difference |
| `style.css` | CSS only used here to VISUALLY demonstrate block vs inline |

## Key Difference
| Block Elements | Inline Elements |
|---|---|
| Start on a NEW line | Stay on the SAME line |
| Take full width | Take only needed width |
| You can set width/height | Cannot set width/height directly |
| `div`, `p`, `h1-h6`, `ul`, `table` | `span`, `a`, `img`, `strong`, `em`, `b`, `i` |
